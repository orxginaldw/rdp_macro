

### rdp_macro

> RDP wrapper based on [SniperThrilla's video](https://www.youtube.com/watch?v=4Oexj5zf84I)

---

### Features


| Option                  | Description                                                                                                      |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------- |
| **Install / Uninstall** | Installs or removes RDPWrap.                                                                                     |
| **Execute Command**     | Whatever you typed runs as **cmd /c …** inside **RDP**.                                                          |
| **Show / Hide Preview** | RDP window like [mstsc.](https://learn.microsoft.com/en-us/windows-server/administration/windows-commands/mstsc) |


---

### Example

Path must be readable by **RDP**:

```sh
start "" "C:\Users\RDP\Downloads\Bot.exe"
```
